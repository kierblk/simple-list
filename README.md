# SimpleList

A simple list web app that allows you to create multiple ideaboards each containing multiple lists that each contain multiple tasks. 

## Getting Started

### Installing

```
bundle install
```

### Usage

```
rails server
```

Then, interact with the web app via the specified localhost port.
